# Unsloth Story Generator (Base Model)

This Python script generates creative stories using the [Unsloth](https://github.com/unslothai/unsloth) library. 

The script is designed to batch-generate `N` number of unique stories with varying themes and structures, saving the results directly to a CSV file.

## Features

* **Use any model**: Loads the specified Unsloth model.
* **High Performance**: Uses Unsloth's 4-bit quantization and native inference optimization for faster generation and lower memory usage.
* **Diverse Generation**: Automatically randomizes themes (e.g., Sci-Fi, Fantasy, Noir) and narrative structures (e.g., Linear, Frame Story, Epistolary) for each generation.
* **Deduplication**: Includes basic checks to prevent duplicate titles or identical content.
* **CSV Export**: Automatically saves generated stories to `generated_stories.csv` with metadata (diversity score, word count).

## Prerequisites

* **Hardware**: A GPU with CUDA support (NVIDIA T4, V100, A100, etc.) is highly recommended.
* **OS**: Linux (or WSL2 on Windows).

## Installation

1.  **Install Unsloth**:
    Follow the official [Unsloth installation guide](https://github.com/unslothai/unsloth) for your specific CUDA version. Generally, for a Colab-like environment or standard CUDA 12.1 setup:

    ```bash
    pip install "unsloth[colab-new] @ git+[https://github.com/unslothai/unsloth.git](https://github.com/unslothai/unsloth.git)"
    pip install --no-deps "xformers<0.0.27" "trl<0.9.0" peft accelerate bitsandbytes
    ```

2.  **Install Python Dependencies**:
    ```bash
    pip install torch pandas transformers
    ```

## Usage

1.  **Configure the Script**:
    Open the script file (e.g., `story_generator.py`) and adjust the configuration variables at the top if needed:

    ```python
    # Number of stories to generate
    N_STORIES = 140 
    
    # Model to use (Must be an Unsloth supported model)
    BASE_MODEL_NAME = "unsloth/llama-3-8b-Instruct-bnb-4bit"
    ```

2.  **Run the Script**:
    ```bash
    python story_generator.py
    ```

3.  **View Results**:
    The script will output progress to the console. Once finished, find the output file `generated_stories.csv` in the same directory.

## Output Format

The generated CSV file contains the following columns:

* **title**: The title generated by the model.
* **text**: The full body of the story.
* **theme**: The random theme selected for this story.
* **structure**: The narrative structure applied.
* **diversity_score**: A calculated ratio of unique words to total words.
* **word_count**: Total word count of the story.
* **score**: Default placeholder (0).

## Troubleshooting

* **Out of Memory (OOM)**: If you run out of GPU memory, try lowering the context length in `load_base_model` (e.g., change `max_seq_length = 2048` to `1024`).
* **Download Issues**: The first time you run the script, it will download the model weights from Hugging Face. Ensure you have a stable internet connection.
